# 📈 Stock Market Analytics Dashboard

Interactive stock market analytics dashboard built with Python.

## Features
- US Stocks: Apple, Microsoft, Tesla
- Indian Stocks: Reliance, TCS, Nifty 50
- Price history with moving averages
- Daily returns analysis
- Rolling volatility
- Correlation heatmap
- Risk vs Return visualization

## Tech Stack
Python · Pandas · NumPy · Plotly · Streamlit · yfinance

## Run Locally
pip install -r requirements.txt
streamlit run dashboard/app.py
